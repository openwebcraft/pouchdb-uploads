# PouchDB Uploads

A little sample application for uploading files to a [PouchDB][pouchdb.com].

## Building Blocks

- HTML5
- [jQuery][jquery.com] (v1.9.1)
- [Underscore.js][underscorejs.org] (v.1.4.4)
- [PouchDB][pouchdb.com]
- [mustache.js][mustache.js]
